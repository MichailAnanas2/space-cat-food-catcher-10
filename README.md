# Космические животные уворачиваются от бомб / Space Animals Dodge Bombs

Playable аркада на чистом HTML5 Canvas и vanilla JavaScript: уворачивайтесь от падающих космических бомб, собирайте звёзды и еду, открывайте 10 милых животных-героев.

## Запуск

1. Откройте `game/space-cat-food-catcher-10/index.html` в современном браузере.
2. Или запустите локальный сервер из папки игры:

```bash
cd game/space-cat-food-catcher-10
python -m http.server 8000
```

и откройте `http://localhost:8000`.

## Управление

**ПК:**
- `←` / `→` или `A` / `D` — движение
- `↑` / `W` / `Пробел` — прыжок / уворот
- `Esc` / `P` — пауза

**Мобильные устройства:**
- Свайп влево/вправо в игровой зоне — движение
- Тап — прыжок
- Опциональные виртуальные кнопки внизу экрана

## Содержимое

- `index.html` — разметка экранов и HUD
- `style.css` — адаптивные стили, палитра, мобильная раскладка
- `game.js` — игровой движок, механика, управление, звук
- `assets/` — растровые спрайты (ai-grok-assets), фоны, UI, звуки
- `data/` — `raster-asset-map.json` и `object-size-map.json`
- `README.md` — этот файл
- `CODE_CREDITS.md` — атрибуция стороннего кода и аудио
